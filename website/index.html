// 02 — TECH STACK

Technologies Used

A complete DevOps stack for modern cloud deployment workflows.

🔴

Jenkins

Open-source automation server that powers our CI/CD pipeline. Configured with a declarative Jenkinsfile to orchestrate every build and deployment.

CI/CD AUTOMATION
☁️

Microsoft Azure

Cloud platform hosting the application via Azure App Service. Scales automatically and provides global availability with 99.9% uptime SLA.

CLOUD HOSTING
🐳

Docker

Containerized the web application for consistent builds. The same Docker image runs in Jenkins tests and gets shipped to Azure — no environment surprises.

CONTAINERIZATION
🐱

GitHub

Source control with webhook integration. Every push to the main branch triggers the Jenkins pipeline automatically — true continuous delivery.

SOURCE CONTROL
🌐

HTML/CSS/JS

This website — built with pure HTML, CSS, and JavaScript. Served via Nginx inside a Docker container for fast, lightweight delivery.

FRONTEND
🐳

Nginx

High-performance web server serving static files inside the container. Configured to handle routing and compression for optimal load times.
